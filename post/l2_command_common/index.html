文件及Linux常用命令

2019.05.30

这是最基础的Linux命令，也是最常用的Linux命令


文件类型及权限
在前面的博客“初识Linux”中已经对于文件类型有了相关说明和解释，在这里补充一个“隐藏文件”，隐藏文件是一种以.开头命名的文件，隐藏文件一般都是在Linux中扮演比较核心作用的文件，不让轻易改动的文件。

文件类型及权限：利用 ls-l 命令我们可以在界面上看到类似-rw-r--r--这样的字眼，在这串代码中就暗含了文件类型信息和权限信息，下面对于这串代码有一定的说明。
以-rw-r--r--为例：
        这一串代码中的第一个字符表示文件类型，-代表它是一个二进制文件、如果为 d 代表它是一个目录、l则表示为软链接文件；
        后面的九个字符被分为三组，分别为rw-、r--、r--这样三组，第一组表明了u(所有者)的权限、第二组表明了g(所属组)的权限、第三组则表明了o(其他人)的权限，在Linux系统中，每个文件都把用户分为这三类，u(user)、g(group)、o(other)；
        r(read)表示读操作、w(write)表示写操作、x(execute)表示执行操作、-表示无任何操作，不难看出，对于以上这个示例，u(所有者)有读和写的操作，却没有执行的操作、g(所属组)仅有读操作、o(其它人)也是仅有读操作。

利用ls-l命令查看相关信息中，在-rw-r--r--后面还有一些信息，如-rw-r--r-- 1 root root 1569 May 27 02:37 anaconda -ks.cfg,它们分别代表了什么呢？
        在-rw-r--r--后面的第一root表示所有者，这里表示所有者是root、第二个root表示所属组，这里表示所属组也为root、1569代表文件的大小，也可以用ls-h命令来用KB、MB、GB等wenidows常见单位来显示大小、May 27 02:37 是在说文件的最后修改时间、anaconda-ks.cfg则是文件名；

每一个文件都有一个id(索引号)：ls-i-xx(inode)命令可以查看索引号为xx的文件,i节点可以相同，如软链接文件的i节点和原文件的i节点标识一样，系统就是通过i节点相同来对软链接文件进行同步更新；而且每一个进程都有一个标识号，每一个用户也都有用户的标识，都是一个数字，root可以通过这个数字来对它们进行管理。

常用Linux命令
文件搜索命令
    1、命令名称find
        所在路径：/bin/find
        执行权限：所有用户
        功能描述：文件搜索
        语法：find [搜索范围] [匹配条件]
           -type 根据文件类型查找：-f 文件、-d 目录、 -l软链接文件
           -inum 根据i节点查找
           -amin -xx(time) 查找在设定访问时间(access)内的文件
           -cmin -xx(time) 查找在设定时间内修改文件属性(change)的文件
           -mmin 查找文件内容
           -name 按照文件名查找、-iname不区分大小写
           示例：find /etc -name init 在目录/etc中查找文件init
           -size 按照文件大小查找文件
           示例：find / -size +204800 在根目录下查找大于100MB的文件
           -user\group 按照用户属性查找文件
           示例：find /home -user sunwukong 在根目录下查找所有者为sunwukong的文件
           -a 满足两个条件、 -o 两个条件满足任意一个即可
           示例：find /etc +163840 -a -size -204800 在/etc下查找大于80MB且小于100MB的文件
           -exec\ok 命令{} \; 对搜索结果执行操作，exec不询问、ok会询问是否确定
           示例：find /etc -name inittab -exec ls -l {} \; 在/etc下查找inittab文件并显示其详细信息
    2、命令名称grep
        所在路径：/bin/grep
        执行权限：所有用户
        功能描述：在文件中搜寻字串匹配的行并输出
        语法：grep -iv [指定字串] [文件]
           -i 不区分大小写
           -v 排除指定字串
           示例：grep mysql /root/install.log 在root/install.log文件中搜寻包含mysql字串的行并输出
    3、命令名称locate
        所在路径：/usr/bin/locate
        执行权限：所有用户
        功能描述：在文件资料库中查找文件
        语法：locate 文件名
           示例：locate inittab
    4、命令名称whereis
        所在路径：/usr/bin/whereis
        执行权限：所有用户
        功能描述：搜索命令所在目录及帮助文档所在目录
        语法：whereis [命令名称]
           示例：whereis ls
    5、命令名称which
        所在路径：/usr/bin/which
        执行权限：所有用户
        功能描述：搜索命令所在目录及别名信息
        语法：which [命令名称]
           示例：which ls
帮助命令
    1、命令名称help
        所在路径：Shell内置命令
        执行权限：所有用户
        功能描述：获得Shell内置命令的帮助信息
        语法：help [命令名称]
           示例：help umask 查看umask命令的帮助信息
    2、命令名称man (manual)
        所在路径：/usr/bin/man
        执行权限：所有用户
        功能描述：获得帮助信息
        语法：man [命令或配置文件]
           示例：man ls 查看ls命令的帮助信息
           示例：man services 查看配置文件services的帮助信息
挂载命令
    命令名称mount
        所在路径：/bin/mount
        执行权限：所有用户
        功能描述：设置文件名 挂载点
        语法：mount [-T 文件系统] 设备文件名 挂载点
           示例：mount -t iso9660 /dev/sr0/mnt/cdrom
压缩解压命令
    1、命令名称zip
        所在路径：/usr/bin/zip
        执行权限：所有用户
        功能描述：压缩文件或目录
           压缩后的文件格式：.zip
        语法：zip 选项[r] [压缩后的文件名] [文件或目录]
             -r 压缩目录
    2、命令名称unzip
        所在路径：/usr/bin/unzip
        执行权限：所有用户
        功能描述：解压.zip的压缩文件
        语法：unzip [压缩文件]
           示例：unzip test.zip
    3、命令名称gzip (GNU zip)
        所在路径：/bin/gzip
        执行权限：所有用户
        功能描述：压缩文件
        语法：gzip [文件]
           压缩后的文件格式：.gz
    4、命令名称tar
        所在路径：/bin/tar
        执行权限：所有用户
        功能描述：打包文件/解压缩文件
        语法：tar 选项 [zcf] [压缩后文件名] [目录] (打包)
             -c 打包
             -v 显示详细信息
             -f 指定文件名
             -z 打包同时压缩
           压缩后的文件格式：.tar.gz
        语法：tar 选项 [压缩文件] (解压文件)
             -x 解包
             -v 显示详细信息
             -f 指定解压文件
             -z 解压缩
           示例：tar -zxvf saheshang.tar.gz
    5、命令名称bzip2
        所在路径：/usr/bin/bzip2
        执行权限：所有用户
        功能描述：压缩文件
        语法：bzip2 选项 [-k] [文件]
           压缩后的文件格式：.bz2
           示例：bzip2 -k sunwukong
              tar -cjf zubajie.tar.bz2 China

    1、命令名称who
        所在路径：/usr/bin/who
        执行权限：所有用户
        功能描述：查看登录用户信息
        语法：who
           示例：who
    2、命令名称w
        所在路径：/usr/bin/w
        执行权限：所有用户
        功能描述：查看登录用户详细信息
        语法：w
           示例：w
    3、命令名称useradd
        所在路径：/usr/sbin/useradd
        执行权限：root
        功能描述：添加新用户
        语法：useradd 用户名
           示例：useradd sunwukong
    4、命令名称passwd
        所在路径：/usr/bin/passwd
        执行权限：所有用户
        功能描述：设置用户密码
        语法：passwd 用户名
           示例：passwd sunwukong
权限管理命令
    1、命令名称chgrp (change file group ownership)
        所在路径：/bin/chgrp
        执行权限：所有用户
        功能描述：改变文件或目录的所属组
        语法：chgrp [用户组] [文件或目录]
           示例：chgrp zhangdhso sunwukong 改变文件sunwukong的所属组为zhangshao
    2、命令名称chmod (change the permissions mode of a file)
        所在路径：/bin/chmod
        执行权限：所有用户
        功能描述：改变文件或目录权限
        语法：chmod [{ugoa} {+-=} {rwx}] [文件或目录] [mode=421] [文件或目录] -R 递归修改
           示例：chmod g+w testfile 赋予文件testfile所属组写权限
              -R 777 testdir 修改目录testfile及其目录下文件为所有用户具有全部权限
          权限的数字表示r ---- 4； w----2； x----1； rwxrw-r-- 7 6 4
    3、命令名称chown (change file group ownership)
        所在路径：/bin/chown
        执行权限：所有用户
        功能描述：改变文件或目录的所有组
        语法：chgrp [用户] [文件或目录]
           示例：chown zhangshao sunwukong 改变sunwukong的所有者为zhangshao
    4、命令名称umask (the user file-creation mask)
        所在路径：Shell内置命令
        执行权限：所有用户
        功能描述：显示、设置文件的缺省权限
        语法：umask [-S] -S 以rwx形式显示新建文件缺省权限
           示例：umask -S
网络命令
    1、命令名称ifconfig (interface configure)
        所在路径：/sbin/ifconfig
        执行权限：root
        功能描述：查看和设置网卡信息
        语法：ifconfig 网卡名称 IP地址
           示例：ifconfig eth0 192.168.8.250
    2、命令名称write
        所在路径：/usr/bin/write
        执行权限：所有用户
        功能描述：给用户发信息，以Ctrl+D保存结束
        语法：write <用户名>
           示例：write zhangshao
    3、命令名称wall (write wall)
        所在路径：/usr/bin/wall
        执行权限：所有用户
        功能描述：发广播信息
        语法：wall [message]
           示例：wall Zhangshao is a honest man ！
    4、命令名称traceroute
        所在路径：/bin/traceroute
        执行权限：所有用户
        功能描述：显示数据包到主机间的路径
        语法：traceroute
           示例：traceroute www.QQ.net
    5、命令名称setup
        所在路径：/usr/bin/setup
        执行权限：root
        功能描述：配置网络
        语法：setup
           示例：setup
    6、命令名称ping
        所在路径：/bin/ping
        执行权限：所有用户
        功能描述：测试网络连通性
        语法：ping 选项 IP地址
            -c 指定发送次数
           示例：ping 192.168.1.156
    7、命令名称last
        所在路径：/usr/bin/last
        执行权限：所有用户
        功能描述：列出目前与过去登入系统的用户信息
        语法：last
           示例：last
    8、命令名称lastlog
        所在路径：/usr/bin/lastlog
        执行权限：所有用户
        功能描述：检查某特定用户上次登录的时间
        语法：lastlog
           示例：lastlog
              lastlog -u 502
    9、命令名称mail
        所在路径：/bin/mail
        执行权限：所有用户
        功能描述：查看发送电子邮件
        语法：mail [用户名]
           示例：mail root
    10、命令名称netstat
        所在路径：/bin/netstat
        执行权限：所有用户
        功能描述：显示网络相关信息
        语法：netstat [选项]
            -t TCP协议
            -u UDP协议
            -l 监听
            -r 路由
            -n 显示IP地址和端口号
           示例：netstat -tlun 查看本机监听的端口
            -an 查看本机所有的网络连接
            -rn 查看本机路由表
关机重启命令
    1、命令名称shutdown
        执行权限：root
        功能描述：实现关机重启
        语法：shutdown -[选项] [时间](可以是具体时间或者分钟数)
            -c 取消前一个关机命令
            -h 关机
            -r 重启
           示例：shutdown -h now 现在关机
    2、退出登录命令logout
    3、其它关机命令halt
        poweroff
        init 0
    4、其它重启命令reboot
        init 6
    5、附：可以用命令runlevel命令查询系统运行级别，cat /etc/inittab 修改系统默认运行级别
        系统运行级别：
               0 关机
               1 单用户
               2 不完全多用户，不含NFS服务
               3 完全多用户
               4 未分配
               5 图形界面
               6 重启
