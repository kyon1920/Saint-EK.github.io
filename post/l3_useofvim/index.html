Vim的使用技巧总结


什么是vim？
vi是运行在Linux/UNIX上的一种老式的文本处理器，工作在字符模式下且不需要图形界面，它的效率很高，不逊色于任何最新的文本编辑器。
vim是从vi发展出来的一款功能强大的全屏幕文本编辑器，是Linux/UNIX上最广泛使用的文本编辑器，它的作用是建立、编辑、显示文本文件，它还具有代码补全、编译及错误跳转等方便编程的功能。

vi/vim的基本概念
vi/vim被分为三种状态：
命令模式(command mode)
    在用户刚刚启动vi/vim，便进入了命令模式，它的主要功能是控制屏幕光标的移动，字符、字或行的删除，移动复制某区段及进入插入模式(insert mode)，或者到底行模式(last line mode)，在此状态下的击键操作将被vim识别为命令，而非输入字符，以下是命令模式下常用的几个命令：
     ▪ i 切换到插入模式，以输入字符
     ▪ x 删除当前光标所在处的字符
     ▪ : 切换到底行命令模式，以在最底一行输入命令
插入模式(insert mode)
    在命令模式下按下i就进入了输入模式；
    只有在插入模式下，才可以做文字/字符的输入，按 [ESC] 键可回到命令行模式；
    在输入模式下，可以键入下列按键进行相关操作：
     ▪ 字符按键以及Shift组合，输入字符
     ▪ ENTER，回车键，换行
     ▪ BACK SPACE，退格键，删除光标前一个字符
     ▪ DEL，删除键，删除光标后一个字符
     ▪ 方向键，在文本中移动光标
     ▪ HOME/END，移动光标到行首/行尾
     ▪ Page Up/Page Down，上下翻页
     ▪ Insert，切换光标为输入/替换模式，光标将变成竖线/下划线
     ▪ ESC，退出输入模式，切换到命令模式
底行模式(last line mode)
    将文件保存或退出vi，也可以设置编辑环境，如寻找字符串、列出行号......
    在命令模式下按下：(英文冒号)就进入了底线命令模式
    在底线模式中，基本的命令有：
     ▪ q 退出程序
     ▪ q! 不保存退出程序
     ▪ w 保存文件
     ▪ 以上命令的组合
    按[ESC]键退出底线命令模式
vim没有菜单，只有命令

vim使用技巧

    导入命令执行结果 :r !命令
    定义快捷键 :map 快捷键 触发命令
        范例： :map ∧P I#
        范例： :map ∧B 0x
    连续行注释
         :n1,n2s/∧/#/g
         :n1,n2s/∧/#//g
         :n1,n2s/∧/\/\//g
    替换： :ab mymail zhangshao976@qq.com
    移动光标的方法
        h 或 向左箭头键(←) 光标向左移动一个字符
        j 或 向下箭头键(↓) 光标向下移动一个字符
        k 或 向上箭头键(→) 光标向右移动一个字符
        l 或 向右箭头键(↑) 光标向上移动一个字符
    vim插入命令
        命令：a 作用：在光标所在字符后插入
        命令：A 作用：在光标所在行尾插入
        命令：i 作用：在光标所在字符前插入
        命令：I 作用：在光标所在行行首插入
        命令：o 作用：在光标下插入新行
        命令：O 作用：在光标上插入新行
        命令：:n1,n2d 作用：删除指定范围的行
    vim复制和剪切命令
        命令：yy 作用：复制当前行
        命令：nyy 作用：复制当前行以下n行
        命令：dd 作用：剪切当前行
        命令:ndd 作用：剪切当前行以下n行
        命令：p、P 作用：粘贴在当前光标所在行下或行上
    vim定位命令
        命令： :set nu 作用：设置行号
        命令： :set nonu 作用：取消行号
        命令： gg 作用：到第一行
        命令： G 作用：到最后一行
        命令： nG 作用：到第n行
        命令： :n 作用：到第n行
        命令： $ 作用：移至行尾
        命令： 0 作用：移至行首
    vim删除命令
        命令：x 作用：删除光标所在处字符
        命令：nx 作用：删除光标所在处后n个字符
        命令：dd 作用：删除光标所在行，ndd删除n行
        命令：dG 作用：删除光标所在行到文件末尾内容
        命令：D 作用：删除光标所在处到行尾内容

本篇博客只是记录了vim的常见操作，其实vim功能之强大，本文未提及的留作后面讨论学习。
