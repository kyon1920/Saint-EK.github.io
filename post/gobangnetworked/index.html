联机对战版-红蓝五子棋

2019.03.02

经典小游戏：童年的五子棋-联网对战版



项目名称：联机对战版-红蓝五子棋
开发工具：VS2019
开发语言：C/C++、windowsAPI、windows网络编程
项目介绍：可以在一台电脑内进行单机游戏，不必开启服务器端，只需要开启一个客户端即可进行游戏；只要两台电脑在同一局域网内，也可以在两台电脑上进行游戏，游戏分为两方，一为红方，一为蓝方，任意一方先下棋，另一方继续下，直到某一方率先筹齐五子连珠者胜。

客户端实现同上次五子棋，服务器端使用了前面的博客中的windows网络编程基础；

服务器端的实现
1、实现方法：由基础的windows网络编程完成，主要知识点是前面windows网络编程博客中涉及的知识。

2、实现逻辑：服务器端的工作为进行客户端通信的中间桥梁，建立两个客户端Socket分别代表客户端1和客户端2，利用一个死循环进行收发消息，总是先收消息，发给客户端2，接收消息，发给客户端1，直到游戏结束，终止条件成立利用break语句结束循环。

3、实现分析：
    a)待优化处：收发消息太过规律，不能应对变化，就例如某一客户端不小心在棋盘区域点了两下鼠标，发送了两次消息，服务器便不能处理，客户端也会卡死；只能以字符或者字符串的形式进行收发消息，不能实现用位置结构体、类对象、或者其它原子类型来实现客户端之间的通信。
    b)优点：能基本实现联机对战五子棋算是一个优点吧。
    c)客户端与服务器端互通位置信息形式：将整型位置信息首先转化成为字符型，然后传送给服务器端，经服务器端发送给另一个客户端，在另一个客户端进行再一次转换，将接收到的字符信息转换为位置信息并于该位置放置棋子。

客户端的实现
1、游戏棋盘：15X15正方形网格图形，由windowsAPI-Rectangle()函数实现最外层框图，由windowsAPI-MoveToEx()和LineTo()两个函数结合循环画线实现内部正方形网格现，该实现被放在PAINT消息里面。

2、实现逻辑：定义一个15x15的二维数组，并将他们全部初始化为0，由鼠标左键实现在棋盘中放置红棋，当玩家在相应位置放置红棋时，向服务器发送一个位置消息，服务器接收到这个位置消息后边立即向另一个客户端发送该位置消息，另一个客户端接收到该位置消息，在对应的位置放置蓝色棋子，轮到对方放置棋子，放置棋子后也是会发送位置消息给客户端，循环往复，红棋对应数组元素设置为1，蓝棋为另一个客户端放置的棋子，对应该数组元素设置为2，每次下棋后(无论是红棋还是蓝棋)，都要进行该处是否被对方棋子占据和五子连珠检查，该棋盘格子处未被占据并且未形成五子连珠方可继续在该处放置棋子，当该位置被对方棋子占据时，另选位置放置棋子即可，若是形成了五子连珠，胜负决出，游戏结束。

3、实现分析：
    a)待优化处：应将棋盘的实现、判断五子连珠单独写在一个函数中，这样可以避免代码重复，减少工作量；由于没有网络模型的支持，客户端只能由两个实体来实现，有一点差异性，也就是有一个客户端第一个动作是发送消息，另一个客户端第一个动作是接收消息，应该避免这样的差异性，所以这是一个待优化的地方。
    b)优点：每次利用鼠标点击来放置棋子时，并不是鼠标点哪里就在哪里放置棋子，而是首先对鼠标点击位置进行判断，只有当鼠标点击位置在棋盘格子的左右上下15个像素点的时候，才会在棋盘放置以该格子为中心，半径为15的圆形相应颜色的棋子，从而避免了鼠标点哪里就在哪里放置棋子。
    c)客户端与服务器端互通位置信息形式：将整型位置信息首先转化成为字符型，然后传送给服务器端，经服务器端发送给另一个客户端，在另一个客户端进行再一次转换，将接收到的字符信息转换为位置信息并于该位置放置棋子。

    }
}

void ConnectServer()
{
    //链接服务器
    struct sockaddr_in sockServerMsg;
    sockServerMsg.sin_family = AF_INET;
    sockServerMsg.sin_port = htons(12345);
    sockServerMsg.sin_addr.S_un.S_addr = inet_addr("127.0.0.1");

    if (SOCKET_ERROR == connect(socketServer, (const sockaddr*)& sockServerMsg, sizeof(sockServerMsg)))
    {
        //出错了
        int a = WSAGetLastError();
        closesocket(socketServer);
        WSACleanup();   //清理/关闭网络库
    }
}

}

void AcceptSocket()
{

void CloseSocket()

int APIENTRY wWinMain(_In_ HINSTANCE hInstance,
                     _In_opt_ HINSTANCE hPrevInstance,
                     _In_ LPWSTR    lpCmdLine,
                     _In_ int       nCmdShow)
{
    UNREFERENCED_PARAMETER(hPrevInstance);
    UNREFERENCED_PARAMETER(lpCmdLine);

    // TODO: 在此处放置代码。

    // 初始化全局字符串
    LoadStringW(hInstance, IDS_APP_TITLE, szTitle, MAX_LOADSTRING);
    LoadStringW(hInstance, IDC_MY, szWindowClass, MAX_LOADSTRING);
    MyRegisterClass(hInstance);

    // 执行应用程序初始化:
    if (!InitInstance (hInstance, nCmdShow))
    {
        return FALSE;
    }

    HACCEL hAccelTable = LoadAccelerators(hInstance, MAKEINTRESOURCE(IDC_MY));

    MSG msg;

    while (GetMessage(&msg, nullptr, 0, 0))

    }

    return (int) msg.wParam;
}



//

//  目标: 注册窗口类。

ATOM MyRegisterClass(HINSTANCE hInstance)
{
    WNDCLASSEXW wcex;

    wcex.cbSize = sizeof(WNDCLASSEX);

    wcex.style          = CS_HREDRAW | CS_VREDRAW;
    wcex.lpfnWndProc    = WndProc;
    wcex.cbClsExtra     = 0;
    wcex.cbWndExtra     = 0;
    wcex.hInstance      = hInstance;

// “关于”框的消息处理程序。

{
    UNREFERENCED_PARAMETER(lParam);
    switch (message)
    {
    case WM_INITDIALOG:
        return (INT_PTR)TRUE;

    case WM_COMMAND:
        if (LOWORD(wParam) == IDOK || LOWORD(wParam) == IDCANCEL)
        {
            EndDialog(hDlg, LOWORD(wParam));
            return (INT_PTR)TRUE;
        }
        break;
    }
    return (INT_PTR)FALSE;
}

//打开网络库
void OpenSocket()
{
    nRes = WSAStartup(wsVersion, &wdScokMsg);   //打开网络库

    if (0 != nRes)  //打开失败
    {
        switch (nRes)
        {
        case WSASYSNOTREADY:
            printf("系统配置存在问题，请重启电脑，或者检查网络库\n");
            break;
        case WSAVERNOTSUPPORTED:
            printf("请更新网络库\n");
            break;
        case WSAEPROCLIM:
            printf("请尝试关闭不必要的软件\n");
            break;
        case WSAEFAULT:    //程序员问题:参数传入错误
            break;
        case WSAEINPROGRESS:
            printf("请重新启动软件\n");
            break;
        }
    }

    if (2 != HIBYTE(wdScokMsg.wVersion) || 2 != LOBYTE(wdScokMsg.wVersion)) //校验版本
    {
        printf("该网络库版本不存在\n");
        WSACleanup();   //关闭网络库
    }

    socketServer = socket(AF_INET, SOCK_STREAM, IPPROTO_TCP);   //创建网络库
    //...相关操作
    if (INVALID_SOCKET == socketServer)    //失败，对错误码进行操作
    {
        int a = WSAGetLastError();    //找出离它最近的socket函数的错误码
        //...相关操作
        WSACleanup();   //清理/关闭网络库
    }
}

void ConnectServer()
{
    //链接服务器
    struct sockaddr_in sockServerMsg;
    sockServerMsg.sin_family = AF_INET;
    sockServerMsg.sin_port = htons(12345);
    sockServerMsg.sin_addr.S_un.S_addr = inet_addr("127.0.0.1");

    if (SOCKET_ERROR == connect(socketServer, (const sockaddr*)& sockServerMsg, sizeof(sockServerMsg)))
    {
        //出错了
        int a = WSAGetLastError();
        closesocket(socketServer);
        WSACleanup();   //清理/关闭网络库
    }
}

//给服务器发送消息
void SendSocket()
{
    if (SOCKET_ERROR == send(socketServer, strBuf1, sizeof(strBuf1), 0))
    {
        //发送失败
        int a = WSAGetLastError();
        //...进行相关操作
        printf("向服务器发送消息失败!");
    }
}

//接收服务器发来的消息
void AcceptSocket()
{

    }
}

void CloseSocket()
{
    //清理网络库
    closesocket(socketServer);    //释放SOCKET
    WSACleanup();
}

ATOM MyRegisterClass(HINSTANCE hInstance)

                {

                    CloseSocket();
                }
            }
        }
        SelectObject(hdc, hOldPen);
        SelectObject(hdc, hOldBrush);

        return 0;

    case WM_PAINT:
    {
        PAINTSTRUCT ps;
        hdc = BeginPaint(hWnd, &ps);

    case WM_DESTROY:

        break;
    default:
        return DefWindowProc(hWnd, message, wParam, lParam);
    }
    return 0;
}

// “关于”框的消息处理程序。
INT_PTR CALLBACK About(HWND hDlg, UINT message, WPARAM wParam, LPARAM lParam)
{
    UNREFERENCED_PARAMETER(lParam);
    switch (message)
    {
    case WM_INITDIALOG:
        return (INT_PTR)TRUE;

    case WM_COMMAND:
        if (LOWORD(wParam) == IDOK || LOWORD(wParam) == IDCANCEL)
        {
            EndDialog(hDlg, LOWORD(wParam));
            return (INT_PTR)TRUE;
        }
        break;
    }
    return (INT_PTR)FALSE;
}
